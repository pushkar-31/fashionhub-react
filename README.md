# FashionHub

A modern Fashion E-Commerce Frontend built using React.

## Features

* Responsive Navbar
* Product Listing
* Product Categories
* Product Search
* Product Details Page
* Shopping Cart
* Quantity Management
* Remove From Cart
* Grand Total Calculation
* Local Storage Persistence
* Responsive Design

## Tech Stack

* React
* React Router DOM
* Context API
* CSS3
* Local Storage

## Folder Structure

src/
├── components
├── pages
├── context
├── data
├── App.jsx
└── main.jsx

## Installation

```bash
git clone <repository-url>
cd fashionhub-react
npm install
npm run dev
```
